== Description ==

The Google Buzz button easily allows your blog to be shared. 


Tested upto 3.0


#### Functionality

* New - Google buzz button. Integrates the ability to share your post with one click.


== Installation ==

Follow the steps below to install the plugin.

1. Upload the google buzz directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in wp
3. Go to "Google Buzz" option to configure the button
